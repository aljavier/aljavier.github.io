Virtualenv: Entornos virtuales de desarrollo para Python

Públicado el Fri 16 August 2013 por A. J. Javier

Instalación

Debian/Ubuntu 

apt-get install virtualenv

Arch Linux 

pacman -S virtualenv

Se creo el directorio donde guardar todos los entornos virtuales:

~/virtualenv

Para crear un entorno virtual cualquiera:

virtualenv ~/virtualenv/nombre
